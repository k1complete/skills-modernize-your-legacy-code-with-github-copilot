# 学生アカウント管理システム - テストプラン

**プロジェクト：** 学生アカウント管理システム（COBOL）  
**作成日：** 2026年3月27日  
**対象システム：** main.cob, operations.cob, data.cob  
**目的：** ビジネスロジックと実装の検証、Node.js移行前の機能検証

---

## テストスコープ

このテストプランは、以下の領域をカバーします：

- **初期化・起動テスト**：システムの正常な起動と初期値設定
- **残高表示機能**：データプログラムからの正確な残高読取
- **入金機能**：正常な入金処理と残高更新
- **出金機能**：正常な出金、残高不足検出、エラーハンドリング
- **メニュー操作**：メニュー表示、入力処理、ループ制御
- **終了処理**：プログラムの正常な終了
- **データ整合性**：READ/WRITE操作による残高の一貫性
